# Caso 7 — Ajuste de cantidad (ejemplo: Inventario físico)

**¿Qué es?** Se hace un conteo físico de la bodega y las cantidades reales no coinciden con lo que dice el sistema.

## Pasos

1. El operador realiza el conteo físico de los productos en bodega.
2. Registra en el sistema las cantidades reales encontradas.
3. El sistema compara el conteo contra el stock registrado.
4. Se generan los ajustes necesarios:
   - Si hay **menos** de lo esperado → ajuste de reducción.
   - Si hay **más** de lo esperado → ajuste de aumento.
5. Cada ajuste queda registrado con fecha, responsable y motivo.
6. A partir de ese momento el inventario refleja la realidad de la bodega.

## Ejemplo concreto

- **Azúcar**: sistema dice 50 kg, bodega tiene 45 kg → ajuste de −5 kg.
- **Harina**: sistema dice 30 kg, bodega tiene 33 kg → ajuste de +3 kg.

---

## Consideraciones

- Los ajustes de cantidad **sí modifican el stock disponible**.
- No modifican el costo registrado en los lotes existentes.
- Quedan visibles en el historial de movimientos del producto para auditoría.
